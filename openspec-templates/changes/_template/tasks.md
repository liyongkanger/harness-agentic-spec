## 任务级循环

每个任务必须按以下闭环完成：

```text
计划 -> 实现 -> 局部验证 -> 任务审查 -> 反思 -> 修正 -> 再验证 -> 再审查 -> done
```

审查失败时，必须在任务的“修正记录”中写清：

- 失败原因
- 修正计划
- 下一轮验证方式
- 修正后审查结论

- [ ] 如出现修正轮次，已记录失败原因和修正证据。
