# 审查矩阵

完成一个需求变更后，用这个清单做最终审查。

## 需求覆盖

- [ ] 每条验收标准都有实现证据。
- [ ] 每个非目标都没有被误做。
- [ ] 每个开放问题都已解决，或被记录为风险。

## 按变更类型检查代码

Controller/API：

- [ ] 请求字段已校验。
- [ ] 返回契约保持兼容，或已明确记录变更。
- [ ] 错误路径已覆盖。

Service：

- [ ] 主流程已覆盖。
- [ ] 异常分支已覆盖。
- [ ] 涉及事务/幂等时已检查影响。

Mapper/Data：

- [ ] SQL 条件符合预期行为。
- [ ] Entity/DTO/XML 字段映射完整。
- [ ] 空值和默认值处理已检查。

Job/MQ：

- [ ] 重跑/重试行为已检查。
- [ ] 重复执行/幂等行为已检查。
- [ ] 失败处理已检查。

## 验证

- [ ] 已尝试编译命令。
- [ ] 已尝试相关测试，或明确说明无法执行。
- [ ] 剩余风险已列出。
